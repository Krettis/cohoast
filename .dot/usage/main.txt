usage: cohoast optional: [ add ] [ block ] [ backup ] [ remove ] | [-h | --help] | [-v | --version ]
Only calling 'cohoast' will take you to a friendly menu
where you can specify a lot more options for handling
your hostfile.

List of commands:
add             Add a hostname to the hostsfile
block           Add hostname to the blacklist
backup          Do a backup of the current hostsfile
remove          Remove a hostname from the hostsfile

-h | --help     Get more information
                how to use a command

-v | --version  Current version in use
