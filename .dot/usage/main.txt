usage: cohoast optional: [ [add ] [ block ] [ backup ] [-h | --help] [-v | --version ]
Only calling 'cohoast' will take you to a friendly menu
where you can specify a lot more options for handling
your hostfile.

List of commands: 
add             Add a hostname to the localhost
block           Add hostname to the blacklist
backup          Do a backup of the current hostfile

-h | --help     Get more information about
                how to use a command

-v | --version	Current version in use
